**cosmodrome** is an image pipeline that produces images.
## Quick Start

### Installation

Install with Go:

```bash
go install github.com/robolaunch/cosmodrome@latest
```

Install binary:

```bash
wget ...
```

### Usage

For example, building Freecad, run:

```bash
cosmodrome launch --config pipelines/freecad.yaml
```
